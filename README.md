# Go Playground

一个轻松、可爱、适合新手的浏览器版围棋入门小游戏。玩家执黑，AI 执白，可以直接在浏览器中打开游玩，不需要后端或复杂框架。

## 功能介绍

- 支持 9 路、13 路、19 路棋盘切换，默认 19 路
- 玩家执黑，AI 执白
- 支持落子、气、提子、禁止自杀和简单打劫
- 支持悔棋、Pass、认输、重新开始
- 双方连续 Pass 后结束游戏
- 按中国规则粗略计算胜负：棋子数 + 围空数
- 简单 AI：优先选择可提子的合法点，否则随机落子
- 动态表情棋子：根据棋组气数显示开心、担心、惊恐
- 同组棋子有柔软连接效果，帮助新手理解棋组
- 手机和桌面浏览器均可使用

## 本地运行方式

下载或克隆仓库后，直接双击打开 `index.html` 即可游玩。

也可以使用任意静态文件服务器预览，例如：

```bash
python3 -m http.server 8765
```

然后在浏览器中打开：

```text
http://127.0.0.1:8765/
```

## GitHub Pages 访问方式

仓库开启 GitHub Pages 后，将使用 `main` 分支发布。

访问地址格式：

```text
https://用户名.github.io/go-playground/
```

实际地址会在 GitHub Pages 开启后更新。
